Sistema de Transporte para Estación de Picking

Control de Banda Transportadora mediante Joystick

Concepto del Proyecto

Banda transportadora a escala diseñada para simular una estación de picking, permitiendo el control preciso de la posición del producto.

Materiales Principales

  • Motor paso a paso NEMA 17
  • Driver A4988
  • Arduino UNO
  • Joystick analógico
  • Fuente de 12V

Prototipo de la Banda Transportadora

Código Arduino

#define DIR 2
#define STEP 3
#define ENABLE 4
#define JOYSTICK A0

int zonaMuertaMin = 400;
int zonaMuertaMax = 624;

void setup() {
  pinMode(DIR, OUTPUT);
  pinMode(STEP, OUTPUT);
  pinMode(ENABLE, OUTPUT);
  digitalWrite(ENABLE, HIGH);
}

void loop() {
  int lectura = analogRead(JOYSTICK);

  if (lectura < zonaMuertaMin) {
    digitalWrite(ENABLE, LOW);
    digitalWrite(DIR, LOW);
    moverMotor(500);
  } else if (lectura > zonaMuertaMax) {
    digitalWrite(ENABLE, LOW);
    digitalWrite(DIR, HIGH);
    moverMotor(500);
  } else {
    digitalWrite(ENABLE, HIGH);
  }
}

void moverMotor(int delayMicros) {
  digitalWrite(STEP, HIGH);
  delayMicroseconds(delayMicros);
  digitalWrite(STEP, LOW);
  delayMicroseconds(delayMicros);
}

El sistema ofrece precisión, robustez y escalabilidad.
